This responsive web application has been created as a part of assessment for DATABASE MANAGEMENT SYSTEMS Lab, 4th Semester, 2017.
The technologies used are -

  • Python 3.5 - Server Side Programming Language
  • flask - Backend Microframework in Python
  • flask_pymongo - MongoDB Client for Python
  • MLab - Free Online MongoDB Sandbox
  • Bootstrap - Frontend Framework
  • Heroku - Free Online Hosting Service
I would like to thank my friend Nikhil Kumar for sharing his knowledge of Python with me. Also, I would like to thank Bucky Roberts and Pretty Printed for their tutorials on Bootstrap and flask/flask_pymongo respectively.

Currently, there are only 15 entries in the database - and user can make queries based on student's full name, or on the basis of the CGPA (which I personally do not like).
